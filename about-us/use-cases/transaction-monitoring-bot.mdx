---
title: "Transaction monitoring bot"
---

You can monitor smart contracts through Events (we only supports EVM blockchains currently).

For example, you can create the following scenarios:

- Monitor all buy and sell orders in the [KEKIUS](https://dexscreener.com/ethereum/0xfff8d5fff6ee3226fa2f5d7d5d8c3ff785be9c74) pool, calculate the USD volume of each transaction, and instantly send a Telegram notification if any buy order exceeds \$50,000.
- You can track the sending and receiving of tokens for any wallet address. Whenever that wallet sends or receives tokens, you'll receive a notification on Telegram.
