---
title: "Other features"
---

This section includes the following features:

- Connect Telegram to receive notifications when running workflows, or to remotely control the execution of campaigns and workflows through your Telegram account.
- Configure API key for Juptiter swap, OpenAI
- Configure the maximum number of workflows that can run simultaneously (to prevent overloading your computer if it has limited resources and too many workflows are running at once).
- Sync your data, import and export all KeeperAgent data for backups or when switching computers
