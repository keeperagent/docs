title: "Trading AI gent"
---

We built an AI agent to simplify trading across multiple wallets with simple prompts. Here are some features the agent can perform:

1. Launch tokens on pump.fun and bonk.fun
2. Create thousands of wallets from prompts
3. Buy any token on Solana or EVM through Jupiter or Kyberswap aggregators, with various swap strategies, for example:
   1. _Buy each wallet 0.5 SOL of token 9BB6NFEcjBCtnNLFko2FqVQBq8HHM13kCyYcdQbgpump_: the agent will buy with the same amount for each wallet.
   2. _Buy minimum 0.1 SOL and maximum 0.5 SOL of token 9BB6NFEcjBCtnNLFko2FqVQBq8HHM13kCyYcdQbgpump_: the agent will buy with random amount depending on wallet balance.
   3. _Buy total 10 SOL of token 9BB6NFEcjBCtnNLFko2FqVQBq8HHM13kCyYcdQbgpump_: the agent will buy with a random amount distributed across all wallets.
   4. _Buy total \$50 of token 9BB6NFEcjBCtnNLFko2FqVQBq8HHM13kCyYcdQbgpump_: The agent converts \$50 to SOL (based on current SOL price) and executes trades across all wallets with random amounts.
   5. _Sell 30% of token 9BB6NFEcjBCtnNLFko2FqVQBq8HHM13kCyYcdQbgpump_: the agent will calculate the total amount of the token across all wallets and sell 30% of each wallet with a random amount distributed across all wallets.
