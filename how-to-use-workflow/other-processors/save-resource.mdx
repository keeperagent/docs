## Configuration fields

| Field          | Required | Description                                                                                                                                                                                 |
| -------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Resource group | Yes      | Pick the Resource group where you want to store the resource. For example, if your Twitter asset group defines Username and Password columns, this processor lets you populate those values |
| Secret key     | No       | If you plan to encrypt resource details before saving them on local database, use this secret key - only you know it                                                                        |
