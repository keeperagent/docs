## Configuration fields

| Field           | Required | Description                                                               |
| --------------- | -------- | ------------------------------------------------------------------------- |
| Search text     | Yes      | Choose the value you want to look up                                      |
| Resource group  | Yes      | Select the Resource group you want to search                              |
| Field to search | Yes      | Which field in that Resource group do you want to search                  |
| Secret key      | No       | If your asset data is encrypted locally, provide the secret key to search |
