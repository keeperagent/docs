## Configuration fields

| Field           | Required | Description                                                                           |
| --------------- | -------- | ------------------------------------------------------------------------------------- |
| Search text     | Yes      | Pick the value to search for, such as the wallet address, seed phrase, or private key |
| Wallet group    | Yes      | Select the wallet group you want to search                                            |
| Field to search | Yes      | Do you want to search by Wallet address, Seed phrase, or Private key                  |
| Secret key      | No       | If your wallet records are encrypted locally, provide the Secret key to search them   |
