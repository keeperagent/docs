This processor offers two execution modes:

- `Without browser`: the JavaScript snippet runs inside Node.js's [VM](https://nodejs.org/api/vm.html) with no browser involvement, so pick this when your code doesn't need to interact with browser.
- `Run in browser`: the JavaScript executes inside Chrome DevTools, launching the browser so you can interact with page elements, for example `document.querySelector(".title").innerHTML`

## Configuration fields

| Field name              | Required? | Detail                                                                                                                                   |
| ----------------------- | --------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| Variable to save result | Yes       | <p> This processor outputs a random number, so select a variable to capture it.</p>  <p> For example <code> EXECUTED_RESULT</code>  </p> |
| Mode                    | Yes       | <code> Without browser</code>   or <code> Run in browser</code>                                                                          |
| Javascript code         | Yes       | The code snippet you want to execute                                                                                                     |
