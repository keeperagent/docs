---
title: "Processor Random On/Off"
description: "Randomly enable or disable workflow execution"
---

Use this processor to add randomness when your workflow runs. For example, if you have 100 profiles, you can randomly select 70% to execute, meaning 70 profiles run and 30 are skipped.

## Configuration fields

| Field          | Required | Description                                                                                                                               |
| -------------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| Percentage (%) | No       | Percentage of profiles allowed to run out of the total, set it to 0 to prevent any profile from running and stop the workflow immediately |
