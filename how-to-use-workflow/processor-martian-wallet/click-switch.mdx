title: "Processor Click Switch"
description: "Click Switch button in Martian wallet"
---

This processor clicks the **Switch** button in Martian. No configuration needed, just drag it into the workflow.

Martian supports both Aptos and Sui. For example, if the wallet is currently on Sui but you need to connect to a dApp that requires Aptos, it will prompt you to switch networks.
