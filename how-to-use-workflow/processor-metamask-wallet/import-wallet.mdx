## Configuration fields

| Field       | Required | Description                                                                                                                     |
| ----------- | -------- | ------------------------------------------------------------------------------------------------------------------------------- |
| Seed phrase | Yes      | The wallet's seed phrase is stored in each profile's default `WALLET_PHRASE` variable, so you can access it directly from there |
| Password    | Yes      | This is the password used to unlock the Metamask wallet                                                                         |
