---
title: "Processor Get gas price"
---

This processor fetches the current average gas price on EVM, Aptos, and Sui chains.

Configuration details:

| Field name              | Required? | Detail                                                                                                                   |
| ----------------------- | --------- | ------------------------------------------------------------------------------------------------------------------------ |
| Variable to save result | Yes       | This processor outputs the chain's current average gas price. Pick a variable name to store it. For example, `GAS_PRICE` |
| Blockchain type         | Yes       | The chain whose gas price you want to retrieve                                                                           |
| Node endpoint group     | Yes       | Select the node provider to interact with the blockchain                                                                 |
