---
title: "Processor Get token price"
---

This processor allows you to retrieve token prices from [CoinGecko](https://coingecko.com/) or [Dexscreener](https://dexscreener.com/)

Configuration details:

| Field name              | Required? | Detail                                                                                                                          |
| ----------------------- | --------- | ------------------------------------------------------------------------------------------------------------------------------- |
| Variable to save result | Yes       | This processor will return the token price in USD. Choose a variable name to store this result.<br />For example: `TOKEN_PRICE` |
| Data source             | Yes       | Choose to retrieve prices from either CoinGecko or Dexscreener                                                                  |

| Field name    | Required? | Detail                                                                                                                      |
| ------------- | --------- | --------------------------------------------------------------------------------------------------------------------------- |
| Chain ID      | Yes       | The blockchain's Chain ID, which you can look up on the [ChainList](https://chainlist.org/?search=Base) website<br /><br /> |
| Token address | Yes       | The address of the token for which you want to retrieve the price                                                           |
