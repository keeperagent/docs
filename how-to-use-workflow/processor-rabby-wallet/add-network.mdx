## Quick-add from ChainList

If the blockchain is already listed on [ChainList](https://chainlist.org/), use this option to add it to Rabby without manually filling fields like `Network name` , `RPC URL`.

## Configuration fields

| Field    | Required | Description                                        |
| -------- | -------- | -------------------------------------------------- |
| Chain ID | Yes      | Chain ID of the blockchain network you want to add |

Configuration details:

## Configuration fields

| Field              | Required | Description                                        |
| ------------------ | -------- | -------------------------------------------------- |
| Chain ID           | Yes      | Chain ID of the blockchain network you want to add |
| Network name       | Yes      | This maps to the Network name field in Rabby       |
| RPC URL            | No       | This maps to the RPC URL field in Rabby            |
| Currency symbol    | No       | This maps to the Currency symbol field in Rabby    |
| Block explorer URL | No       | This maps to the Block explorer URL field in Rabby |
