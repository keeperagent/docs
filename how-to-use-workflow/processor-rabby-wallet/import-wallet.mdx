## Configuration fields

| Field       | Required | Description                                                                                                                            |
| ----------- | -------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| Seed phrase | Yes      | The wallet's seed phrase is stored in each profile's default `WALLET_PHRASE` variable, so you can access it directly from there        |
| Private key | Yes      | It's the wallet's private key. Every profile includes a default variable that stores it, so just use the `WALLET_PRIVATE_KEY` variable |
| Password    | Yes      | This is the password used to unlock the Rabby wallet                                                                                   |
