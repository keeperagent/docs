## Configuration fields

| Field                                 | Required | Description                                                                                                                                                                                                                               |
| ------------------------------------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Customize Gas limit?                  | No       | Set a custom gas limit for the transaction                                                                                                                                                                                                |
| Customize Gas price?                  | No       | Set a custom gas price for the transaction                                                                                                                                                                                                |
| Skip warnings and choose "Ignore all" | No       | For contracts you haven't interacted with, new or uncommon ones Rabby shows a warning with an `Ignore all` option. Enable that setting and the processor will automatically click `Ignore all` so it can continue signing the transaction |
