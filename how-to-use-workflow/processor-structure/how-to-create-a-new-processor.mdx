## Creating a new Processor

You can create a new Processor by dragging and dropping from the Processor panel into the workflow editor.

<div align="center">
<img src="/images/how-to-create-a-new-processor-image-1.png" alt="Drag and drop to create a new Processor" style={{ maxWidth: "600px" }} />

<p style={{ fontSize:"0.875rem",marginTop:"-1rem",color:"#666" }}>
Drag and drop to create a new Processor

</p>

</div>

## Connecting Processors

**Step 1**: To connect Processors, hover your mouse over the source Processor to reveal a purple arrow icon.

<div align="center">
<img src="/images/how-to-create-a-new-processor-image-2.png" alt="Hover to reveal the purple arrow" style={{ maxWidth: "600px" }} />

<p style={{ fontSize:"0.875rem",marginTop:"-1rem",color:"#666" }}>
Hover over the `Uniswap swap` Processor to reveal the arrow icon.

</p>

</div>

**Step 2**: _Hover over the purple arrow_, drag it, and then drop it onto the target Processor you want to connect to

<div align="center">
<img src="/images/how-to-create-a-new-processor-image-3.png" alt="Drag the arrow to connect Processors" style={{ maxWidth: "600px" }} />

<p style={{ fontSize:"0.875rem",marginTop:"-1rem",color:"#666" }}>
Drag from the `Uniswap swap` Processor and drop onto the `Bot message` Processor

</p>

</div>

Once connected, the source Processor will execute first, followed by the destination Processor.
