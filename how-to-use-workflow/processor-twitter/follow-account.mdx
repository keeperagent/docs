## Configuration fields

| Field       | Required | Description                                           |
| ----------- | -------- | ----------------------------------------------------- |
| Account URL | Yes      | The URL of the account you want to Follow or Unfollow |
| Action type | Yes      | Follow or Unfollow                                    |
