## Configuration fields

| Field       | Required | Description                                              |
| ----------- | -------- | -------------------------------------------------------- |
| Tweet URL   | Yes      | The URL of the tweet you want to ReTweet or Undo ReTweet |
| Action type | Yes      | ReTweet or Undo ReTweet                                  |
